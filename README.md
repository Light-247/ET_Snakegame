# ET Snakegame 🐍✨

넘버블록스에서 영감을 받은, 아이와 함께 즐기는 숫자 스네이크 게임입니다. 별을 먹으며 블록을 길게 만들고, 숫자별 스페셜 능력과 퍼즐 모험을 즐겨 보세요.

## 주요 기능

- 숫자가 커질수록 달라지는 귀여운 넘버블록 캐릭터
- 싱글 플레이와 2인 협동 플레이
- 안전하게 즐기는 **안 죽는 모드**, 도전 모드, 퍼즐형 모험 모드
- 3·4·5·8·10 블록의 스페셜 능력
- 한국어/영어 음성 안내(TTS), 배경 음악, 7가지 테마
- 별을 모아 여는 넘버블록 도감
- `P` 키 또는 화면 버튼으로 일시정지

## 실행 방법

별도의 설치 과정이 필요 없는 정적 웹 게임입니다.

1. 이 저장소를 내려받습니다.
2. `index.html`을 브라우저에서 열거나, 아래 명령으로 로컬 서버를 실행합니다.

```bash
python3 -m http.server 4173
```

3. 브라우저에서 `http://127.0.0.1:4173`을 엽니다.

## 조작법

| 기능 | 조작 |
| --- | --- |
| 플레이어 1 이동 | 방향키 또는 화면 D-pad |
| 플레이어 2 이동 (협동 모드) | `W` `A` `S` `D` |
| 스페셜 능력 | `Space` |
| 일시정지 / 계속하기 | `P` 또는 화면 버튼 |

## 게임 모드

| 모드 | 설명 |
| --- | --- |
| 안 죽는 모드 🛡️ | 벽을 통과하며 부담 없이 숫자를 쌓아요. |
| 도전 모드 💥 | 벽과 몸에 부딪히지 않도록 집중해요. |
| 모험 모드 🗺️ | 목표 길이를 만들고, 수학 게이트와 퍼즐을 풀어 포탈로 탈출해요. |

## 기술

- HTML5 Canvas
- Vanilla JavaScript
- CSS
- Web Audio API 및 Speech Synthesis API

## 프로젝트 파일

- `index.html` — 화면 구조와 UI
- `styles.css` — 반응형 스타일과 테마
- `game.js` — 게임 규칙, 캔버스 렌더링, 모험 모드
- `audio.js` — 효과음, 배경 음악, 음성 안내
